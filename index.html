Bienvenido a mi página web

Esta página te permitira tomar los datos de tablas de impuestos y convertirlas en formato .json.:

  • Registrar Tablas
  • Visualizar Tablas

Secciones:

Registrar Tablas: Esta sección te permite tomar los datos de todas las tablas publicadas en un año y guardarlas en un solo archivo JSON.

Visualizar Tablas: Esta sección te permite visualizar los datos de un archivo JSON como tablas.
